# Code of Conduct

## Our Pledge
We as contributors and maintainers pledge to respect all people, create a welcoming environment, and engage constructively.

## Expected Behavior
- Be friendly, courteous, and patient.
- Refrain from demeaning or discriminatory comments.
- Provide constructive feedback.
- Respect privacy and consent.
- Follow project standards for coding and communication.

## Unacceptable Behavior
- Harassment, bullying, or discrimination in any form.
- Personal attacks or inflammatory language.
- Sharing private information without permission.
- Spamming, trolling, or disruptive behavior.

## Reporting Issues
If you experience or witness unacceptable behavior, please report it to the project maintainers via email or the project's communication channels. Reports will be handled confidentially.

## Enforcement
Project maintainers may remove contributors or block access to the project if behavior violates this code of conduct. Decisions will be made to protect the community and maintain a safe environment.
